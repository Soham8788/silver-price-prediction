# Silver Price Prediction

## 📌 Overview
This project predicts silver prices using historical data and machine learning models (Linear Regression & Random Forest).

## 🚀 Features
- Fetches live silver price data from Yahoo Finance
- Data preprocessing and feature engineering
- Multiple ML models for prediction
- Streamlit web app for interactive predictions

## 📊 Models Used
- Linear Regression
- Random Forest Regressor
